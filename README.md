*🚀 Instalación rápida*

Compatible con *Linux*, *Termux* y *Ubuntu*:

```bash
# Instala Git y Node.js
```
apt install git -y
apt install nodejs -y
```
```
# Clona el repositorio
git clone https://github.com/AzamiJs/Mini_WaBot.git
cd Mini_WaBot
```
# Instala dependencias
```
npm install
```
# Ejecuta el bot
```
node index.js
```
